# SQL Data Warehouse and Analytics Project

Welcome to the **Data Warehouse and Analytics Project** as part of the SQL Full Course from ***Data with Baara***

This project demonstrates the design and implementation of a modern data warehouse using SQL Server running in Docker. The solution consolidates sales data from multiple sources to enable analytical reporting and business insights.

### Building the Data Warehouse (Data Engineering)

### Objectives
- Data Sources: Import data from two sources (CRM and ERP) provided as CSV files
- Clean and transform raw data into analytical models
- Implement a scalable data warehouse design
- Enable reporting and business insights
